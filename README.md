NUMBER GUESSING GAME 
Solution for Number Guessing Game project from roadmap.sh.

Welcome to the Number Guessing Game. It is a simple command-line game. All of the action takes place in the terminal interacting with the game via keyboard. Computer generates a random number between 1 and 100 and the player must guess it in order to win!

How to run? 
Clone this repository: 

How to play? 
After cloning the repository you have to run the game via IDE or terminal and follow the instructions wihtin the game. 

Difficulty levels
After welcoming message the game offers 3 difficulty levels:
1. Easy - The player has 10 attempts to guess the generated number.
2. Medium - THe player has 5 attempts to guess the generated number.
3. Hard - The player has 3 attempts to guess the generated number. 

Requirements
The game was built in the Python 3.13.7 with Python "random" library. So to run it the user requires:
- Python (version 3.13.7 or higher)
- Python "random" package
